  • git init : create new local repository

  • git add (filename) : add one or more file to staging
  • git commit -m "commit message" : commit changes to head (not yet to repository)
  • git push origin master : send changes to master branch of your repository
  • git status : list the files you've changed and those you still need to commit
  • git remote add origin (Server) : add the server to be able to push to it
  • git pull : fetch and merge changes on the remote server to your working directory.
